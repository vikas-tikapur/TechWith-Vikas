🔸Common Tuple Methods :

  • count(value) : Returns the number of times a value appears in the tuple.
  • index(value): Returns the index of the first occurrence of the value.
  • len() : Returns the number of items in the tuple.
  • max() and min() : Returns the largest and smallest item in the tuple.
  • sum() : Returns the sum of all elements (must be numbers).
  • sorted() : Returns a sorted list of the tuple's elements.
  • tuple() : Converts a list or other iterable to a tuple.
  • Tuple Unpacking : Allows you to assign the elements of a tuple to multiple variables.
  • Asterisk(*) : Used to unpack elements from a tuple.
  • join() : Joins elements of a tuple into a single string.
  • Multiply : You can multiply a tuple by an integer to repeat its elements.
  • remove(value): Removes the first occurrence of the value from the tuple (creates a new tuple).
  • Range of Indexes : You can access a range of items in a tuple using slicing.
  • Negative Indexing : You can access items from the end of the tuple using negative indices.

🔸count(value)

fruits = ("apple", "banana", "cherry", "banana")
print(fruits.count("banana"))  # Output: 2

🔸index(value):

fruits = ("apple", "banana", "cherry")
print(fruits.index("banana"))  # Output: 1

🔸len() :

fruits = ("apple", "banana", "cherry")
print(len(fruits))  # Output: 3

🔸max() and min() :

my_tuple = (5, 10, 25, 2)
print(max(my_tuple))  # Output: 25
print(min(my_tuple))  # Output: 2

🔸sum() :

my_tuple = (5, 10, 25, 2)
print(sum(my_tuple))  # Output: 42

🔸sorted() :

my_tuple = (5, 10, 25, 2)
print(sorted(my_tuple))  # Output: [2, 5, 10, 25]

🔸tuple() :

my_list = [5, 10, 25, 2]
my_tuple = tuple(my_list)
print(my_tuple)  # Output: (5, 10, 25, 2)

🔸Tuple Unpacking :

my_tuple = (5, 10, 25, 2)
a, b, c, d = my_tuple
print(a, b, c, d)  # Output: 5 10 25 2

🔸Asterisk(*) :

my_tuple = (5, 10, 25, 2)
a, b, *c = my_tuple
print(a, b, c)  # Output: 5 10 [25, 2]

🔸join() :

my_tuple = ("apple", "banana", "cherry")
result = ", ".join(my_tuple)
print(result)  # Output: "apple, banana, cherry"

🔸Multiply :

my_tuple = (5, 10, 25, 2)
result = my_tuple * 2
print(result)  # Output: (5, 10, 25, 2, 5, 10, 25, 2)

🔸remove(value):

my_tuple = ("apple", "banana", "cherry", "banana")
my_tuple = tuple(x for x in my_tuple if x != "banana")
print(my_tuple)  # Output: ('apple', 'cherry')

🔸Range of Indexes :

my_tuple = (5, 10, 25, 2)
print(my_tuple[1:3])  # Output: (10, 25)

🔸Negative Indexing :

my_tuple = (5, 10, 25, 2)
print(my_tuple[-1])  # Output: 2
print(my_tuple[-3:-1])  # Output: (10, 25)

🔸When to Use Tuples?

Use tuples when:

  • You have data that should not change
  • You want to store coordinates (x, y)
  • Storing related data (e.g., database records)
  • You want to store person info (name, age, etc.)
  • You want to return multiple values from a function

🔸Summary

  • Tuples are like lists but cannot be changed
  • Use parentheses () to create tuples
  • Access items using square brackets [0], [1], etc.
  • Good for storing data that should stay the same
  • Simple and fast to use!

Difference between List and Tuple:

Feature List Tuple
Syntax list1 = [1, 2, 3] tuple1 = (1, 2, 3)
Mutability Mutable (can be changed) Immutable (cannot be changed)
Methods Many built-in methods (append, remove, etc.) Fewer built-in methods (count, index)
Performance Slower than tuples Faster than lists
Use Case When data may change When data should not change
Brackets Square brackets [ ] Parentheses ( )
Can be used as dictionary key? No Yes (if all elements are immutable)
