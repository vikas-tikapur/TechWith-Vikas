Python Syntax

Python syntax refers to the set of rules that define how a Python program is written and interpreted.

Key Points of Python Syntax:

  • No Semicolons (;) Unlike other languages, Python does not require semicolons to end a line.
print("Hello, World")
  • Indentation is Mandatory Python uses indentation (spaces or tabs) to define blocks of code like loops, functions, or conditionals.
if 5 > 2:
    print("5 is greater than 2")
  • Case-Sensitive Python is a case-sensitive programming language. This means that uppercase and lowercase letters are treated as distinct in identifiers (like variable names and function names). For example, the variable name is different from Name in Python.
name = "Vikas"
Name = "Mishra"
  • Comments Use # for single-line comments.
# This is a comment
print("Hello, Vikas")
  • Variables Don't Need Declaration You can directly assign values to variables without declaring their type.
x = 10
name = "Vikas"
print(x)
print(name)
  • Simple Print Statement Output is printed using print().
print("Welcome to Python!")
  • Function Definition Functions are defined using def keyword.
def greet():
    print("Hello from Python!")
