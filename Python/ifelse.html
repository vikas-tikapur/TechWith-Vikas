if Statement

The if statement in Python is used to make decisions in your code. It allows your program to execute a block of code only if a certain condition is true.

Basic Syntax: 

if condition:
    # code block to execute if condition is true
  • The condition must return True or False.
  • Python uses indentation (usually 4 spaces) to define the block of code.

Example: 

age = 18
if age >= 18:
    print("You are eligible to vote.")     #Output - You are eligible to vote.

üî∏How it works:

  • The condition age >= 18 is checked.
  • If it's True, the code inside the if block runs.
  • If it's False, the code is skipped.

üî∏if with else:

Adds an alternative path when the condition is False.

Syntax: 

if condition:
    # code if True
else:
    # code if False

Example: 

temperature = 25
if temperature > 30:
    print("It's hot")
else:
    print("It's not hot")
# Output: It's not hot

üî∏if-elif-else Statement:

Handles multiple conditions in sequence.

Syntax: 

if condition1:
    # code
elif condition2:
    # code
else:
    # code

Example: 

score = 85
if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
else:
    grade = "D"
print(f"Your grade is {grade}")
# Output: Your grade is B

üî∏Nested if Statements :

if statements inside other if statements.

Example: 

num = 15
if num > 10:
    print("Greater than 10")
    if num % 2 == 0:
        print("Even number")
    else:
        print("Odd number")
# Output: Greater than 10
#         Odd number

üî∏Ternary Operator (Conditional Expression) :

A concise way to write simple if-else statements.

Syntax: 

value_if_true if condition else value_if_false

Example: 

age = 20
status = "Adult" if age >= 18 else "Minor"
print(status)  # Output: Adult

üî∏Logical Operators in Conditions :

Combine multiple conditions using and, or, not.

Example: 

# AND (both must be True)
if age > 12 and age < 20:
    print("Teenager")
# OR (at least one True)
if day == "Saturday" or day == "Sunday":
    print("Weekend")
# NOT (inverts the condition)
if not is_raining:
    print("No umbrella needed")

üî∏Truthy and Falsy Values :

Python treats certain values as implicitly False :

  • False
  • None
  • 0 (any numeric zero)
  • Empty sequences: "", [], (), {}

Example: 

name = ""
if not name:
    print("Name is empty")  # This will execute
