Arithmetic Operations in NumPy

NumPy (Numerical Python) is a powerful library for numerical computing in Python. It provides efficient ways to perform arithmetic operations on arrays.



Supported Arithmetic Operators :


Operator Description Symbol
Addition Adds two arrays +
Subtraction Subtracts arrays -
Multiplication Multiplies arrays *
Division Divides arrays /
Modulus Remainder %
Power Exponentiation **
Floor Division Removes decimals //
