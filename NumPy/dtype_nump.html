NumPy Data Types (dtypes)

In NumPy, data types (also called dtypes) define the kind of values an array can hold, such as integers, floating-point numbers, or strings. Choosing the correct data type is important for performance, memory optimization, and data accuracy.



Why Data Types Matter :

  • NumPy is built for numerical efficiency.
  • Using the right dtype helps process large arrays faster.
  • Ensures precision in scientific and mathematical calculations.


Common NumPy Data Types :


Data Type Description Example
int8 Integer (1 byte) -128 to 127
int32 Integer (4 bytes) -2,147,483,648 to 2,147,483,647
int64 Integer (8 bytes) Very large numbers
float16 Floating point (2 bytes) Less precise float
float32 Floating point (4 bytes) Standard float
float64 Floating point (8 bytes) High precision float
bool Boolean value (True or False) 0 or 1
complex64 Complex numbers (real+imag) 1.5 + 2.3j
str String data "Hello"


How to Check the Data Type of an Array :

Use the dtype attribute to check the data type of a NumPy array:

import numpy as np
arr = np.array([1, 2, 3])
print(arr.dtype)

# Output: int64


Get Data Type of a String Array :

import numpy as np
arr = np.array(["apple", "banana", "cherry"])
print(arr.dtype)

# Output: <U6


Explanation :

1. <U6 means it's a Unicode string of max length 6 characters.

2. NumPy automatically detects the data type based on the input.



Identify the Data Type of a String-Based NumPy Array :

import numpy as np
arr = np.array(["apple", "banana", "cherry"])
print("Array:", arr)
print("Data type:", arr.dtype)

#Output - 
Array: ['apple' 'banana' 'cherry']
Data type: <U6

NumPy dtype Examples (With Code & Output) :

1. b – Boolean

import numpy as np
arr = np.array([True, False, True], dtype='b')
print(arr, "→", arr.dtype)

#Output - 
[ True False True] → bool


2. i – Signed Integer

import numpy as np
arr = np.array([1, -2, 3], dtype='i')
print(arr, "→", arr.dtype)

#Output - 
[ 1 -2  3] → int32


3. u – Unsigned Integer

import numpy as np
arr = np.array([10, 20, 30], dtype='u1')  # uint8
print(arr, "→", arr.dtype)

#Output - 
[10 20 30] → uint8


4. f – Floating Point

import numpy as np
arr = np.array([1.5, 2.7, 3.9], dtype='f')
print(arr, "→", arr.dtype)

#Output - 
[1.5 2.7 3.9] → float32




5. c – Complex Floating Point

import numpy as np
arr = np.array([1.5 + 2j, 3.1 + 4.2j], dtype='c')
print(arr, "→", arr.dtype)

#Output - 
[(1.5+2j) (3.1+4.2j)] → complex64


6. m – Timedelta

import numpy as np
arr = np.array([np.timedelta64(1, 'D'), np.timedelta64(2, 'D')], dtype='m')
print(arr, "→", arr.dtype)

#Output - 
#[1 2] → timedelta64[D]


7. M – Datetime

import numpy as np
arr = np.array([np.datetime64('2023-08-05'), np.datetime64('2023-08-06')], dtype='M')
print(arr, "→", arr.dtype)

#Output - 
#[2023-08-05 2023-08-06] → datetime64[D]


8. O – Object

import numpy as np
arr = np.array([{"name": "Vikas"}, {"name": "John"}], dtype='O')
print(arr, "→", arr.dtype)

#Output - 
#[{'name': 'Vikas'} {'name': 'John'}] → object


9. S – String (Bytes)

import numpy as np
arr = np.array([b'abc', b'def'], dtype='S')
print(arr, "→", arr.dtype)

#Output - 
#[b'abc' b'def'] → bytes


10. U – Unicode String

import numpy as np
arr = np.array(["hello", "world"], dtype='U')
print(arr, "→", arr.dtype)

#Output - 
#["hello" "world"] → unicode


11. V – Void (Raw Data)

import numpy as np
arr = np.array([b'abc', b'def'], dtype='V')
print(arr, "→", arr.dtype)

#Output - 
#[b'abc' b'def'] → bytes
